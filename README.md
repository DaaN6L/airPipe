# AirPipe
## Система контролю аеродинамічної труби.


Код для ознайомлення з ESP32.

Спочатку слід підключити бібліотеку ESP32 до Ардуїно - google esp32 arduino.

Потім завантажити папку з файлами і відкрити файл *.ino в Ардуїно.


### Компоненти, що використовуються:

ESP32 DevKit - https://arduino.ua/prod3990-wi-fi-modyl-devkit-v1-s-esp-32

Тензодатчик - https://arduino.ua/prod1979-datchik-vesa-tenzodatchik-1-kg-visokotochnii

Модуль тензодатчика HX711 - https://arduino.ua/prod1147-hx711-dual-channel-weighing-sensor-module

Датчик тиску BMP280 для трубки Піто - https://arduino.ua/prod1758-barometr-datchik-atmosfernogo-davleniya-na-bmp280

Симісторний регулятор для вентилятора - https://arduino.ua/prod2958-dimmer-220v-dlya-arduino
